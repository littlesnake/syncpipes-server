List PipelineExecutions
GET/pipeline-executions{?limit,page,order}

List all pipeline executions

Example URI

GET http://localhost:3010/api/v1/pipeline-executions?limit=100&page=1&order=started
URI Parameters
HideShow
limit
number (required) Example: 100

The maximum number of pipeline executions which will be in the response.

page
number (required) Example: 1

The page to display, only if the total count of executions is greater then the limit.

order
string (required) Example: started

The field to order the collection, prefix with a minus “-” to order descending.
